# RogueScape Presentation Pack

This folder contains presentation-ready versions of the RogueScape plugin overview.

## Files

- `roguescape-deck.html`  
  Browser-based slide deck. Open it in Chrome/Edge and use it like a lightweight presentation.

- `google-doc-outline.md`  
  Google Docs friendly version. Paste/import this into a Google Doc if you want a shareable written presentation.

## Suggested Use

For a visual walkthrough, open:

```text
docs/presentation/roguescape-deck.html
```

For a planning/share document, use:

```text
docs/presentation/google-doc-outline.md
```
